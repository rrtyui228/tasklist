Создайте веб-приложение, которое будет содержать список задач на React и бэкэндом на Nodejs с использованием сервисов AWS/Azure.

Каждая задача должна иметь возможности:
- Редактирования
- Отметки (не) сделано
- Дату выполнения (или без даты, если не проставлено)

Список задач должен иметь возможность сортировки по сделанным и не сделанным задачам, по задачам, срок выполнения которых крайний к текущей дате, возможность текстового поиска по задачам

Расширенные возможности:
- Синхронизация между офлайн / онлайн (когда задачи создаются без интернета, после подключения, происходит синхронизация)
- Возможность создания профиля пользователя (JWT). Если пользователь не создал профиль, список хранится для "инкогнито" пользователя, т.е. при перезагрузке страницы, список будет получен для этого юзера по его уникальному ключу. Если пользователь создает аккаунт и имел список задач для "инкогнито" пользователя, они добавляются в его профиль.
- Добавить дату выполнения задачи, с возможностью добавления в календарь на устройстве.
- Добавление новых задач должно присылать пуш-уведомление на устройства пользователя *
- Задачи, срок которых подходит к концу, должны присылать пуш-уведомления **
